Терминал Git

Управление репозиториями

  • git init – Инициализация нового или повторная инициализация существующего репозитория
  • git status – Отображение статуса репозитория, отслеживания и изменения файлов
  • git add – Добавление файлов и папок

Управление изменениями

  • git commit – Внесение изменений
  • git log – Просмотр истории всех коммитов
  • git show – Просмотр информации об одном коммите
  • git diff – Просмотр изменений до коммита
  • git restore – Отмена изменений и возвращение файла к состоянию последнего коммита
  • git reset – Отмена любого количества сделанных коммитов

Ветвление

  • git branch – Создание новой ветки
  • git checkout – Перемещение между ветками
  • git branch -d – Удаление ветки
