Терминал Docker

Управление реестрами и репозиториями

  • docker login – Вход в реестр
  • docker logout – Выход из реестра
  • docker search – Поиск образа
  • docker pull – Выгрузка образа из реестра
  • docker push – Загрузка образа в реестр

Действия с контейнерами

  • docker create – Создание контейнера
  • docker run – Первый запуск контейнера
  • docker rename – Переименование контейнера
  • docker rm – Удаление контейнера
  • docker update – Обновление контейнера
  • docker start – Запуск остановленного контейнера
  • docker stop – Остановка контейнера

Управление образами

  • docker images – Отображение списка всех образов
  • docker build – Создание образа
  • docker rmi – Удаление образа
